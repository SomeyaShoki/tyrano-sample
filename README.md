# 走るか寝るかするメロス

ティラノスクリプトで作成した脱出ゲーム集です。

## テストプレイ

- https://appi-github.github.io/tyrano_sample/

## 収録コンテンツ

- 会議室からの脱出（TRPG風・ステージ制）
- 高層ホテルからの脱出

## 実装メモ

- タイトル画面はPC/モバイル判定でレイアウトを切り替え
- 一部UIは画像ではなく動的オブジェクト生成で表示
- セーブ/ロード時の衝突対策として、ロード時に動的UI要素をクリーンアップ
